

# 偉倫外賣後端:

## 欄位說明:

### 	1._id為後端生成,新增時不需填入,後續須使用其操作。

### 	2.__v為後端生成無須理會,後續操作無需使用。

### 	3.其餘值請自行填入,為保證頁面完整性請不要填入空值,竟可能在前台阻擋用戶輸入空值。
